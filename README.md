# Predictive Modeling of Precipitation in South Carolina
## Overview

Developed machine learning models in Python to predict daily precipitation across South Carolina using NOAA historical climate data (2000–2023).

## Skills Demonstrated
- Python
- Machine Learning
- Data Cleaning
- Feature Engineering
- Time-Series Validation
- XGBoost
- Random Forest
- Data Visualization
  
# Key Results
- Compared regression and classification approaches
- Improved prediction performance by reframing the problem
- Identified key climate variables influencing precipitation

## Technologies Used 

- Python 3.10
- Jupyter Notebook
- pandas, numpy, matplotlib, seaborn
- scikit-learn
